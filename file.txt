❯ what is this project all about                                                

⏺ This is Spendly — a personal expense tracker web app built with Flask
  (Python). It's structured as a step-by-step student project.

  Goal: Let users track where their money goes by logging expenses, categorizing
   them, and viewing spending patterns.

  Planned features (when complete):
  - User registration & login/logout
  - Add, edit, and delete expenses (with category, amount, date, description)
  - View spending breakdowns by category and time period
  - User profile page

  Current state — early scaffold:
  - Landing page, login, and register pages have HTML/CSS ready
  - Routes exist but most are placeholders returning strings like "coming in 
  Step X"
  - database/db.py is empty — waiting to be implemented with SQLite (get_db(),
  init_db(), seed_db())
  - No backend logic yet (no auth, no CRUD, no database)

  Tech stack: Flask, SQLite, Jinja2 templates, vanilla CSS/JS

  The project is meant to be built incrementally across ~9 steps, starting with
  database setup and progressing through authentication, expense CRUD, and
  filtering.
